Report Bugs

As with any great piece of software, VR Juggler is likely to have some bugs in it.   Knowing this, the developers have set up a bug reporting service.

This service allows VR Juggler users to report bugs and to suggest improvements.   When submitting reports, please follow the guidelines set forth and give the title of the submission a prefix of "todo:" or "bug:" in order to facilitate sorting of incoming submissions.

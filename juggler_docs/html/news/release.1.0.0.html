April 17, 2001
 VR Juggler 1.0.0 released

The VR Juggler team is proud to make the official release announcement for VR Juggler 1.0.0 (``Wookie''). This version represents fifteen months of enhancement, debugging, and testing since the first open source release of VR Juggler in January 2000. In these fifteen months, we have been fortunate to recieve many bug reports, feature suggestions, and patches from VR Juggler users all over the world.

The full source code and pre-compiled binaries for IRIX 6.5 and Win32 are available from the download page. Please review the RELEASE_NOTES for instructions on upgrading from previous beta versions of VR Juggler. The KNOWN_BUGS file contains a list of all bugs known at the time of release. For the most up-to-date list of bugs, please visit the VR Juggler bug tracker.

Note for RedHat Linux users:

Due to recent complications related to release of GCC 3.0 snapshots with RedHat Linux 7.0, we are unable to prepare a stable binary version of VR Juggler for Linux. It is known that the default version of GCC that ships with RedHad Linux 7.0 will not compile VR Juggler. The recently released update to GCC 2.96.69 can be used to compile the VR Juggler libraries and OpenGL-based VR Juggler applications. Performer-based applications will not run if compiled with GCC 2.96.69 due to problems with C++ constructs. We hope that a future update to RedHat Linux 7.0 will provide a more stable compiler. Users of RedHat Linux 6.1 and 6.2 should experience no problems with VR Juggler using the default compilers. We encourage Linux users to download the source code and compile it locally using the process documented in the README file.

About VR Juggler
VR Juggler is a virtual platform for VR application development. It provides an object-oriented development environment that allows applicatinos to be developed independently of the underlying technologies of the available VR system. VR Juggler's goal is to allow VR applications to be developed once and run anywhere. VR Juggler is designed with extendability and flexibility in mind.

Availability
VR Juggler is available on Irix, GNU-Linux, BSD, Solaris, HP, and Windows NT/2000 platform.  We are currently looking for parties interested in testing VR Juggler on these platforms. We welcome patches and additions (bug fixes, input and output drivers, features, etc...) to the Juggler source. We are setting up a site on sourceforge as a way to bring you the access to the latest source code via CVS, Juggler forum, messaging, and bug tracking. We will make an announcement when the Juggler sourceforge site is fully functional.

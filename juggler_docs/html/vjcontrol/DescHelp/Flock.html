Flock of Birds Chunk

Flock of Birds Chunk token: ``Flock''
Configuration for Ascension Flock of Birds tracking system.
Port Property token: ``port''
The string name of the serial port this device is attached to. On Unix systems, a typical value would be something like ``/dev/ttyd3''.
Baud Rate Property token: ``baud''
The speed of the serial port connection, in bps. Typical values might include 19200 and 38400.
Num Birds Property token: ``num''
The Number of birds in the flock.
Translation Property token: ``translate''
An XYZ translation to be applied to data from this tracker.
Rotation Property token: ``rotate''
XYZ Rotation to be applied to data from this tracker.
Calibration File Property token: ``calfile''
Absolute path name of a tracker calibration file.
Hemisphere Property token: ``hemi''
The hemisphere of the tracker. Choices are FRONT_HEM, AFT_HEM, UPPER_HEM, or LOWER_HEM.
Filter Property token: ``filt''
Filtering method to use. Choices are AC_NARROW, AC_WIDE, and DC_FILTER.
Synchronization Style Property token: ``sync''
See Flock of Birds manual p.82. Choices are NoSync and Normal 72Hz.
Blocking Property token: ``blocking''
Should the port block? 0 for no, 1 for yes.
Report Frequency Property token: ``report''
Report rate. Value should be one of the single character strings ``Q'' (every cycle), ``R'' (every other cycle), ``S'' (every 8 cycles), and ``T'' (every 32 cycles).
Transmitter Property token: ``transmitter''
Integer ID of the Flock's transmitter.

Gesture Proxy Chunk

Gesture Proxy Chunk token: ``GestureProxy''
Defines a pointer to some gesture device. The device pointed to must be one of the known gesture devices.
Device Property token: ``device''
The device to which the proxy points. The proxy points must point to a SimGloveGesture chunk. More chunk types can be added through the chunk description file (chunksDesc).

Gesture Proxy chunk definition

chunk GestureProxy "GestureProxy" "A Gesture Proxy points to a gesture device."
   Name String 1 "Name" "Unique name of an instance of this chunk type"
   device Chunk 1 "device" vj_enumeration { "SimGloveGesture" } "The gesture device to hook the proxy to."
   end

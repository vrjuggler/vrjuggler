Glove Proxy Chunk

Glove Proxy Chunk token: ``GloveProxy''
Defines a pointer to some glove device. The device pointed to must be one of the known glove devices.
Device Property token: ``device''
The glove device to which the proxy points. The proxy points to another chunk that must be in a predefined set of types. Those types are currently:
  • CyberGlove
  • SimGloveGesture
More chunk types can be added through the chunk description file (chunksDesc).
Unit Property token: ``unit''
Sets a unique number for each glove proxy for use when querying the device with the getData() method. The numbering should be non-decreasing starting from 0.
Visible Property token: ``visible''
A flag telling VR Juggler whether or not to render the glove in the scene. It takes a Boolean value of either True or False.

Glove Proxy chunk definition

chunk GloveProxy "GloveProxy" "A Glove Proxy points to a glove device and unit"
   Name String 1 "Name" "Unique name of an instance of this chunk type"
   device Chunk 1 "device" vj_enumeration { "CyberGlove" "SimGloveGesture" } "The glove device to hook the proxy to."
   unit Int 1 "unit" "The unit number in the device to hook to."
   visible Bool 1 "visible" "Should the glove be visible. Should we render it?"
   end

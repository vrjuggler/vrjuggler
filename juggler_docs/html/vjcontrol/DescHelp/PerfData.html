Performance Data Display Chunk

Performance Data Display Chunk token: ``PerfData''
A chunk that describes to vjcontrol how to display performance data from a particular source. Currently very simple, this chunk will be expanded as needed to provide control over visually presenting performance data.
Name Property token: ``name''
The name of the chunk is also the class of performance tests that it provides information about. Examples in the Juggler beta are ``Kernel'' and ``vjGlPipe''.
Labels Property token: ``labels''
Most of the Juggler performance tests divide a particular process, such as the main vjKernel loop, into multiple phases which measured individually. The labels property lets you assign text labels to each phase. For example, the labels for a simplified version of the Kernel loop would be:
  1. Application PreDraw
  2. Application Draw
  3. Swap Buffers
  4. Update Projections

Performance Data Display chunk definition

chunk PerfData "Performance Data Display" 
        "Information for displaying data from perfbuffers with a matching name"
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  labels String -1 "Labels" "Labels for Perf Buffer phases"
  end
